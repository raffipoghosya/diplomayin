Կայունություն

Դիտարկել ԱԱԱՊ և Էկոլոգիական (ESG) ազդեցությունը ծախսերի վրա:

Մեթոդաբանական Հիմքեր

Ֆինանսական Կայունություն

Ֆինանսական անկախության գործակից:

G1 = Սեփական Կապիտալ / Հաշվեկշիռ

Ընթացիկ իրացվելիություն:

G9 = Ընթացիկ Ակտիվներ / Ընթացիկ Պարտավորություններ

Ներդրումների Գնահատում (DCF)

Զուտ Ներկա Արժեք (NPV):

NPV = Σ [ CFt / (1+r)t ] - I0

Որտեղ CF-ը դրամական հոսքերն են, r-ը դիսկոնտի դրույքն է, I0-ն՝ սկզբնական ներդրումը:

Ֆինանսական դրության փոփոխությունը (2023-2024)

  • Սեփական կապիտալն աճել է տպավորիչ 8.9 մլրդ դրամով (կուտակված շահույթի հաշվին):
  • Ընթացիկ պարտավորությունների կտրուկ նվազում -20.9 մլրդ դրամով (շահաբաժինների մարում):
  • Եզրահանգում: Պարտքային բեռը տեղափոխվել է ավելի կայուն՝ երկարաժամկետ հատված:

  • ՈՒՇԱԴՐՈՒԹՅՈՒՆ. Գործող կապիտալի կտրուկ անկում:

import pandas as pd import numpy_financial as npf # Ներդրումային նախագծի մոդելավորում cash_flows = [-9124, 1000.8, 1641.7, 2260.0, 2586.2] discount_rate = 0.10 npv = npf.npv(discount_rate, cash_flows) irr = npf.irr(cash_flows) print(f"NPV: {npv:.2f} մլն, IRR: {irr:.2%}")
