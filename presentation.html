  • Berk-DeMarzo — Մոդելավորում, Ներդրումների Գնահատում
  • Ross-Westerfield-Jaffe — Պարտք/Սեփական Կապիտալ Հարաբերակցության Օպտիմալացում
  • Սելեզնյովա-Իոնովա — Ֆ/Ն Վերլուծություն + WACC Համակարգ
  • Բալաբանով Ի.Տ. — Ֆինանսական Ռիսկերի Կառավարում (Պարտքային Բեռ)

Լիզինգի Տեսական Հիմքերը

Ֆ/Ն Լիզինգ — On-balance → Ամորտիզացիա + Տոկոսավճարներ Հարկվող Շահույթը Նվազեցնում են: Գործառնական Լիզինգ — Off-balance → Սեփական Կապիտալի Ազատում Ներդրումների Համար + Իրացվելիություն ↑:

Երկարաժամկետ Ֆինանսավորման Նպատակները

  • Ոչ Ընթացիկ Ակտիվների Ձեռքբերում + Արդիականացում + Փոխարինում
  • Ֆինանսական Կապիտալի Օպտիմալացում — WACC Մինիմիզացում
  • Վերաֆինանսավորման Ռիսկ — Մարման Պահին Նոր Ֆ/Ն Աղբյուրներ
  • ԴՖԼ Էֆեկտ — ROE Մաքսիմալացում ԵԹԵ ROA > K_d
  • Matching Սկզբունք — Երկարաժամկետ Ակտիվներ ← Երկարաժամկետ Աղբյուրներ
Գործառնական Ֆին. Աղբյուրներ:
Ներքին = Զուտ Շահույթ + Ամորտիզացիա
Արտաքին·Սեփական = Ակցիաներ, Բաժնետոմսեր
Արտաքին·Պարտքային = Բանկային Վարկ, Լիզինգ, Պարտատոմսեր

DP — Բելմանի Օպտիմալության Հավասարում
